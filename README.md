# Stock Price Multilinear Regression

A machine learning project that predicts stock closing prices using multilinear regression.

## Dataset
The dataset contains the following features:
- Open
- High
- Low
- Volume
- MarketSentiment
- Close (target variable)

## Model
- Algorithm: Multilinear Linear Regression
- Library: Scikit-learn
- Train/Test Split: 80/20

## Files
- `StockClosing-multiLinear.ipynb` - Jupyter notebook with the full code
- `stock_dataset.csv` - Dataset used for training and testing
